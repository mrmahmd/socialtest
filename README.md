# الأماكن والاتجاهات من حولنا

تطبيق عربي تفاعلي فاخر للدرس الأول من الوحدة الأولى في الدراسات الاجتماعية للصف الرابع الابتدائي، إعداد المعلمة شهد زهران.

## ما يتضمنه التطبيق

- واجهة افتتاحية بصورة ثلاثية الأبعاد وبيانات الطالبة.
- محطتان تعليميتان تغطيان شرح الدرس كاملًا.
- مخططات تفاعلية للمنزل والمدرسة واختيار الطريق.
- شرح الشمس والنجوم والبوصلة والاتجاهات الأصلية والفرعية.
- بطاقات أسئلة مجابة وتدريبات صح أو خطأ واختيار من متعدد وتوصيل.
- نتيجة فورية وتصميم متجاوب مع الجوال والتابلت والكمبيوتر.

## التشغيل

```bash
pnpm install
pnpm dev
```

ثم افتح `http://localhost:3000`.
